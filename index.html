    What project are you building to solve a real-world problem?

    Project Name: Attendance Management System.

    Tech Stack:

  • Backend: Java (Servlets & JSP)
  • Frontend: HTML, CSS, JavaScript (with optional support for Bootstrap/jQuery)
  • Database: MySQL
  • Authentication & Logic:Java Session Management, AJAX for dynamic updates
  • Key Feature: Face Recognition with Location & Time-based Access Control

    • Project Overview:

    This is a full stack Attendance Management System designed to simplify and digitize the process of marking, tracking, and managing employee or student attendance. The system supports two major roles—User and Admin—both managed within a single, unified application.

    How It Works:

    1. User Side (Employee/Student):

  • Secure login using unique credentials.
  • Attendance is logged only via face recognition at authorized locations and within designated time slots, preventing proxy or remote check-ins.
  • Location and time constraints are pre-set by the Admin (e.g., office or college premises).
  • Users can:
  • View daily attendance.
  • Check monthly attendance summaries.
  • Apply for leave.
  • Edit limited profile data.
  • Submit feedback or requests to Admin.

    • 2. Admin Side:

  • Full access through a secure Admin portal.
  • Admin can:
  • Define valid locations and time slots for attendance using geolocation APIs or IP restrictions.
  • View real-time attendance logs.
  • Add, remove, or update user details.
  • Manage leave requests with approve/reject options.
  • Monitor trends via visual charts (e.g., attendance rates, late entries).
  • All data is persistently stored in MySQL, optimized for speed and integrity.
